Variáveis

var

Esta sintaxe pode ser usada para declarar tanto variáveis locais como variáveis globais.

let

Declara uma variável local de escopo do bloco, opcionalmente, inicializando-a com um valor.

const

Declara uma constante de escopo de bloco, apenas de leitura.

OBS.: Um identificador JavaScript deve começar com uma letra, underline (_), ou cifrão ($); os caracteres subsequentes podem também ser números (0-9). Devido JavaScript ser case-sensitive, letras incluem caracteres de "A" a "Z" (maiúsculos) e caracteres de "a" a "z" (minúsculos).
